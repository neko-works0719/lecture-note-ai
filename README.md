# 講義音声ノートAI

講義の音声ファイルを入れると、文字起こし・要約・要点を自動生成するWebアプリです。

## 使い方

1. アプリをブラウザで開く
2. Gemini APIキーを入力して「保存」を押す（初回のみ）
3. 講義名を入力する（任意）
4. 音声ファイルを選択する
5. 「▶ 文字起こし＆要約を実行」を押す
6. 文字起こし・要約・要点が表示される
7. コピーボタンで内容をコピーできる

## APIキーの取得方法

1. https://aistudio.google.com/apikey を開く
2. 「新しいプロジェクトを作成」でAPIキーを発行
3. アプリの入力欄に貼り付けて保存

## 対応音声形式

MP3 / WAV / AAC / M4A / FLAC / OGG

## 使用技術

- HTML / CSS / JavaScript
- Gemini API（gemini-2.5-flash）

## 注意事項

- APIキーは端末内（localStorage）にのみ保存されます
- 個人情報・機密情報の音声はアップロードしないでください
- 短い音声（30秒〜2分）での利用を推奨します
