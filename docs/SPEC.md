# SPEC.md：講義音声ノートAI 仕様書

## 概要
大学の講義音声を入力すると、文字起こし・要約・要点を自動生成するWebアプリ。
Gemini APIを使用。フロントエンドのみ（HTML/CSS/JS）で実装。

## 使用技術
- HTML / CSS / JavaScript（フロントエンドのみ）
- Gemini API（Google AI）
- 公開：GitHub Pages

## 使用モデル
- 公式モデル一覧を参照：https://ai.google.dev/gemini-api/docs/models?hl=ja
- 音声対応モデルを選ぶ（例：gemini-2.5-flash など最新のもの）

## ユーザー導線（最短）
1. APIキーを入力（初回のみ）→ localStorageに保存
2. 講義名を入力（任意）
3. 音声ファイルを選択（mp3 / m4a / wav など）
4. 「実行」ボタンを押す
5. 文字起こし全文が表示される
6. 要約・要点が表示される
7. それぞれコピーできる

## 画面構成
- APIキー入力エリア（初回のみ表示）
- 講義名入力欄
- 音声ファイル選択ボタン
- 実行ボタン
- 文字起こし表示エリア＋コピーボタン
- 要約・要点表示エリア＋コピーボタン

## エラー処理
- APIキー未入力 → 「APIキーを入力してください」と表示
- ファイル未選択 → 「音声ファイルを選択してください」と表示
- API失敗 → エラー内容をそのまま画面に表示（デバッグのため）

## APIキーの扱い
- コードに直書き禁止
- ユーザーが画面で入力 → localStorageに保存
- GitHubリポジトリには含めない

## 制約・注意
- 最初は短い音声（30秒〜2分）でテストする
- 個人情報・機密の音声は使わない
- スマホでも崩れないレスポンシブデザイン
